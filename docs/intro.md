## 前端学习记录

博主一般不做学习笔记，主要是没有看笔记的习惯，记两页之后就放在角落里吃灰。
但是大学的学习过于碎片化，如果学过的知识不常用，遗忘的很快。所以准备将学习过程记录在这里。

并且最近准备按照前端学习路线重新建立知识体系，尽可能地在原有的知识上深入一些。

:::tip 前端学习路线

所选择的前端学习的次序是根据 GitHub 上 `Kamran Ahmed` 所提供的 [前端学习路线](https://roadmap.sh/frontend)。

:::
